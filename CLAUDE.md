# task-app

三姉妹タスクアプリ Phase 3。Vite + React + Firebase (Firestore + Auth)。

## 作業ルール

- 作業開始前に _STATUS.md を読むこと
- 仕様書: docs/spec-phase3.md
- 指示書: docs/instructions-phase3.md
- 仕様に記載のない判断が必要な場合は停止し報告
- 仕様書・指示書の原本を改変しない
- 中断時は _STATUS.md を更新すること
- 仕様書の§番号単位でコミットすること
- キャラ定義は配列から動的生成すること（ハードコード禁止）

## コミットメッセージ形式

```
[Phase3-§N] 作業タイトル

何を: 実装した内容
なぜ: 仕様書 §N への参照
どのように: 技術的アプローチ
テスト: 実行結果
```

## 環境

- OS: Windows 11
- Shell: PowerShell（`&&` は使わず `;` で連結すること）
- Node.js v24 / npm v11
- Firebase CLI インストール済み・ログイン済み
- Firebase プロジェクト: task-app-6e764
- リポジトリ: https://github.com/misfortunemate-png/task-app
